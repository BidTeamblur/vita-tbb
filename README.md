# VitaTBB
VitaTBB is a small, C++14-compatible subset of oneAPI Threading Building Blocks adapted for PlayStation Vita.

This is an experimental port done mostly by trial and error. Pull requests and fixes are very welcome.

Currently implemented:
- tbb::blocked_range
- tbb::blocked_range2d
- tbb::blocked_range3d
- tbb::blocked_nd_range
- tbb::split / tbb::proportional_split
